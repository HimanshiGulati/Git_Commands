git init  -- To initialise git
git remote add origin <git_path>   -- To add folder or code to created repository
git status     -- To check the status of the files
git add .      -- To add code on git
git commit -m "commit_message" -n    -- To commit lastest changes on git
git push -u origin master    -- To push code at the first time only
git push       -- To push code at the second time

For authentication :--

git config --global user.email "EmailID"
git config --global user.name "username"
